# OpenAI Agents SDK example

This example shows a single-file wallet-analysis agent using the OpenAI Agents SDK and Zerion's REST API.

## Requirements

- `OPENAI_API_KEY`
- `ZERION_API_KEY`
- Python 3.10+
- `uv`

## Run

```bash
export OPENAI_API_KEY="sk-..."
export ZERION_API_KEY="zk_dev_..."
uv run wallet_analysis.py
```
