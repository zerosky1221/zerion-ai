# Cursor example

Use the hosted Zerion MCP in Cursor.

Files:

- `mcp.json`
- `prompt.md`

## Setup

1. Export your API key:

   ```bash
   export ZERION_API_KEY="zk_dev_..."
   ```

2. Add the JSON from `mcp.json` to your Cursor MCP configuration.
3. Paste the prompt from `prompt.md`.
