# Claude example

Use the hosted Zerion MCP in a Claude-compatible MCP client.

Files:

- `mcp.json`
- `conversation.md`

## Setup

1. Export your API key:

   ```bash
   export ZERION_API_KEY="zk_dev_..."
   ```

2. Add `mcp.json` to your Claude MCP setup.
3. Use the prompt flow in `conversation.md`.
